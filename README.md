SmartHttpProxy
==============

智能科学的HTTP代理（反代Socks5）  
进度：开发中  
下面要做的事情  
1、反代socks5  
2、添加2份域名名单，1份名单内域名直接扔给socks5；另一份直接连接，走正常HTTP proxy流程。  
3、解析域名ip，并判断是否在国内ip段内，国内直连，与第二条相同。
